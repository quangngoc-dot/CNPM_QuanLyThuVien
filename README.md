## Web API for Library Management System
**This is my personal project and is not affiliated with any real library.**
This project is a Web API built using ASP.NET Core for managing a library system. It provides endpoints to manage books, authors, and borrowers.
### 📚 Library Web API (ASP.NET Core)
### Features 
- Manage books,users,loan flips and book loan requests
- User authentication via Google Login (OAuth2)
- Connect to the database using Entity Framework Core
- Communicate with FontEnd via RESTful API
### Technologies 
- ASP.NET Core 8.0
- Entity Framework Core
- SQL Server
- Identity + Google OAuth
- JWT Authentication
- CORS cho frontend (React / Vite)
